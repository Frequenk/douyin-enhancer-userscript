# 🎬 网页抖音体验增强

[![Greasy Fork](https://img.shields.io/greasyfork/v/539942?label=Greasy%20Fork)](https://greasyfork.org/zh-CN/scripts/539942-%E7%BD%91%E9%A1%B5%E6%8A%96%E9%9F%B3%E4%BD%93%E9%AA%8C%E5%A2%9E%E5%BC%BA)
[![License](https://img.shields.io/badge/license-GPL--3.0-blue.svg)](LICENSE)

一个强大的油猴脚本，为网页版抖音提供更纯净、更高效的浏览体验。

## ✨ 核心功能

### 1. 🚀 自动跳过
- ⏭️ **直播间**：自动检测并跳过直播内容
- ⏭️ **广告**：自动识别并跳过广告视频

### 2. 🚫 智能屏蔽 ⭐
- 🎯 **多维度屏蔽**：支持按账号名称、视频简介、#话题标签三种方式检测关键字
- ⚙️ **智能处理**：可选"不感兴趣"(R键)告诉算法，或仅跳到下一个视频
- 📁 **导入/导出**：支持通过 .txt 文件批量管理您的屏蔽关键字列表

### 3. 📺 画质优化
- 🔝 **自动最高分辨率**：智能选择当前视频可用的最高分辨率 (4K > 2K > 1080P...)
- 🔒 **锁定4K**：找到4K视频后可自动关闭此功能，避免不必要的切换
- ⚙️ **分辨率筛选**：只观看您指定分辨率的视频，不符合的将自动跳过

### 4. 🤖 AI智能筛选 ⭐ (需本地Ollama)
- 🧠 **内容识别**：自定义您想看的内容（如"风景"、"猫咪"），AI将自动为您筛选
- ❤️ **智能点赞**：当AI判定为您喜欢的内容时，可选择自动点赞(Z键)
- ⚡ **快速决策**：通过多时间点截图检测，实现快速精准判断

### 5. ⚡ 极速模式
- 🕒 **定时切换**：每个视频播放指定时间后自动切换到下一个，适合快速浏览
- ⏱️ **自定义时间**：可在1-60秒内自定义每个视频的播放时长

## 📦 安装

### 前置要求
- 浏览器需安装用户脚本管理器：
  - [Tampermonkey](https://www.tampermonkey.net/)

### 安装方式

**方式一：从 Greasy Fork 安装（推荐）**

[![安装脚本](https://img.shields.io/badge/安装脚本-点击这里-green?style=for-the-badge)](https://greasyfork.org/zh-CN/scripts/539942-%E7%BD%91%E9%A1%B5%E6%8A%96%E9%9F%B3%E4%BD%93%E9%AA%8C%E5%A2%9E%E5%BC%BA)

**方式二：手动安装**

下载 `douyin-enhancer.user.js` 文件，在脚本管理器中导入。

## 🎮 使用方法

1. 打开 [网页版抖音](https://www.douyin.com/)
2. 在播放器控制栏可看到统一控制面板
3. 点击各项功能的标题文字，即可打开专属的详细设置弹窗
4. 所有设置自动保存在浏览器本地

### ⌨️ 快捷键
- `=`：开启/关闭"跳过直播"功能

## 🤖 AI功能配置 (可选)

如需使用AI智能筛选功能，需要：

1. 安装 [Ollama](https://ollama.com/) 并下载视觉模型（默认：`qwen3-vl:8b`）

2. 开启Ollama跨域模式，设置环境变量：
   ```
   OLLAMA_HOST=0.0.0.0
   OLLAMA_ORIGINS=*
   ```

3. 参考配置教程：[Ollama跨域设置指南](https://lobehub.com/zh/docs/self-hosting/examples/ollama)
